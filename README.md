# Salias
A easy installation script for SillyTavern

---

一个简单易用的SillyTavern安装脚本

## 安装

```bash
curl -O https://raw.githubusercontent.com/LimeBlogs/Salias/refs/heads/main/Install.sh && chmod 7777 Install.sh && apt update && apt install fzf -y && ./Install.sh
```

输入以下命令来快速使用。
该脚本会自动切换软件源，以便国内用户使用。

## 额外说明
对于脚本的"9选项"，你需要保证你使用的代理软件是Clash，并且正确设置端口后才能使用

非Clash直接使用可能会导致问题

没遇到问题可以不运行9选项，如果你卡在了拉取源码处，可以考虑使用流量或者关闭梯子，多试试，肯定没问题
